### 内容简介

本书深入探讨了动态资产定价的实证研究方法，旨在为读者提供一套完整的理论框架和计量经济学工具。书中详细介绍了广义矩方法（GMM）等核心数学技术在资产定价模型设定、估计与评估中的应用，并涵盖了各种计量经济学工具及其在实证研究中的具体案例。通过本书，读者将能够系统掌握如何构建、检验和评估动态资产定价模型，从而更好地理解金融市场中的资产行为。

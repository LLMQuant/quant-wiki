# Recommended Books on AI for Quantitative Trading

## *Machine Learning for Trading*

<img src="../images/ml-trading.jpg" alt="Machine Learning for Trading" width="200"/>

- **Author**: Stefan Jansen
- **Publisher**: Packt
- **Year**: 2020
- **Difficulty**: ⭐⭐⭐⭐
- **Recommendation**: ⭐⭐⭐⭐⭐

### Overview

This book provides a detailed guide to applying machine learning in quantitative trading. It covers a broad spectrum of techniques -- from linear models and tree-based models to deep learning and reinforcement learning -- and demonstrates how to apply them to market data, fundamental data, and alternative data sources for building predictive models, designing trading strategies, and performing backtesting and optimization.

### Key Chapters

1. Machine Learning Fundamentals
2. Data Preprocessing
3. Feature Engineering
4. Building Predictive Models
5. Strategy Backtesting and Optimization

### Practical Resources

- Complete code repository
- Sample datasets
- Online supplementary materials

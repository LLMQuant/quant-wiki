## **Quant Mind**: Intelligent Agent Framework for Quantitative Finance

QuantMind is a knowledge extraction and retrieval framework designed for quantitative finance. It helps developers rapidly build AI Agents for the financial domain, transforming unstructured content such as academic papers, research reports, and news into actionable, structured knowledge.

Visit the [GitHub repository](https://github.com/LLMQuant/quant-mind) to learn more.

![QuantMind](../asset/quantmind.png)

???+ success "Key Features"
    * **Agent-Native:** Built-in agentic workflows and tool integrations, supporting the construction of vertical-domain agents for tasks like paper reading and research report analysis.
    * **Context Engineering:** Emphasizes structured processing of financial context, providing high-quality domain knowledge as input for LLMs.
    * **Two-Stage Architecture:** Decoupled design separating knowledge extraction (Source -> Parser -> Knowledge Base) from intelligent retrieval (Embedding -> RAG/DeepResearch).
    * **Multi-Source Ingestion:** Supports multiple data sources including arXiv, news APIs, and financial blogs, with a unified knowledge format standard.
    * **Open-Source & Extensible:** MIT licensed, modular design, easy to customize data sources, retrieval strategies, and agent logic.

## **Quant Paper**: AI-Powered Quantitative Research Paper Platform

**[Quant Paper](https://www.quantpaper.com/)** is an AI-powered platform for searching and reading academic papers in quantitative finance. It hosts a curated collection of 100+ selected quantitative finance papers and supports AI-powered search with multi-dimensional tag filtering.

![quantpaper-dark](../asset/quantpaper-dark.png)

???+ success "Key Features"
    * **AI-Powered Search:** Supports keyword search and an AI Deep Research mode for quickly locating relevant papers.
    * **Multi-Dimensional Tags:** Organized along two dimensions -- market/asset (23 tags) and research topic (56 tags).
    * **Structured Abstracts:** Each paper features structured extraction of key information for efficient reading.
    * **Continuously Updated:** Regularly updated with the latest quantitative finance research.

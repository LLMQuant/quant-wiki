# What Is Present Value?
![](https://fastly.jsdelivr.net/gh/bucketio/img9@main/2024/10/20/1729465031968-b3c8959e-1d37-4b8a-91b1-b0b0dfe25143.png)
Present value (PV) is the current value of a future sum of money or stream of cash flows. It is calculated by discounting the future value at an estimated rate of return. If the money were invested, the discount rate represents the rate of return it could earn. Present value calculations are important in both investment analysis and corporate strategic planning.

### Key Takeaways

- Present value (PV) is based on the concept that a given amount of money today is typically worth more than the same amount in the future, because it can be invested and earn a return in the interim.
- Calculating present value requires an estimate of the potential rate of return, known as the discount rate.
- Since the discount rate is only an estimate, present value calculations can be inaccurate if the estimate is off.
- Investors can also calculate the future value (FV) by applying an estimated rate of return to today's value.

## Understanding Present Value

The concept of present value holds that a specific sum of money today is generally worth more than the same amount in the future -- this is known as the time value of money. Conversely, a specific amount of money receivable in the future is worth less than that amount today.

For example, $1,000 today should be worth more than $1,000 received five years from now, because today's $1,000 can be invested and earn returns over those five years. Assuming an annual return of 5%, compounded annually, that $1,000 would grow to approximately $1,276 in five years.

The present value calculation works in reverse. Suppose you are due to receive $1,000 in five years (the future value, FV). What is that money worth today? At the same annual rate of return of 5%, the answer is approximately $784. In this formula, the rate of return is called the discount rate. The term "discount" refers to the process of translating a future value back to the present.

## Present Value Formula and Calculation

The formula for calculating the present value of a future sum of money is:

$$ \begin{aligned} &\text{Present Value} = \dfrac{\text{Future Value}}{(1+r)^n}\\ &\textbf{where:}\\ &\text{Future Value} = \text{FV}\\ &r = \text{rate of return}\\ &n = \text{number of periods}\\ \end{aligned} $$

To calculate the present value of a future stream of cash flows, repeat this formula for each cash flow and sum the results. Fortunately, modern software and online calculators make this straightforward, eliminating the need for manual computation.

## Determining the Discount Rate

As noted, the discount rate is the rate of return used in a present value calculation. It represents the return you forgo by choosing to receive a sum in the future rather than the same amount today. The discount rate is highly subjective because it is only an estimate of the return you might earn by investing today's money over a period of time.

In many cases, investors use a risk-free rate of return as the discount rate. This typically refers to the yield on U.S. Treasury securities, which are considered virtually risk-free because they are backed by the U.S. government.

The higher the discount rate you choose, the lower the present value, because you are assuming a higher return could be earned on the money.

## Benefits of Present Value

- Present value can clarify whether the expected return on an investment is sufficient to make it worth pursuing. Businesses and investors often set a hurdle rate representing the minimum return required.
- Present value helps investors and companies choose between different investments.

## Limitations of Present Value

- Present value requires assumptions about the discount rate, and these assumptions may not be accurate.
- Because of these assumptions, present value calculations can be manipulated by those who advocate for a particular investment, such as corporate managers pushing for a legacy project.

**Note:** By using a real interest rate rather than a nominal rate, you can also incorporate the potential effects of inflation into the present value formula.

## Present Value Example

As a simple example, suppose you can choose between receiving $2,000 today or $2,200 one year from now. You expect you can safely invest the $2,000 and earn a 3% return. Which option is better?

In this case, $2,200 is the future value (FV), so the present value (PV) calculation is $2,200 / (1 + 0.03)^1. The result is $2,135.92. Therefore, if you want to receive the money now, you would need to be offered at least $2,135.92 -- not just $2,000 -- to break even.

## Calculating Future Value vs. Present Value

In the present value formula above, we assumed you know the future value and need to calculate the present value.

When you know the present value, you can also calculate the future value using the formula: FV = PV x (1 + r)^n.

Plugging in the same numbers from the example above:

FV = $2,000 x 1.03 = $2,060.

Based on both the present value and future value calculations, waiting for the $2,200 one year from now is a better deal than taking $2,000 today.

Of course, both calculations depend on whether the rate of return you select is accurate.

## How Do You Calculate Present Value?

Present value is calculated using three data points: the expected future value, the interest rate available if the money were invested, and the number of payment periods -- for example, 1 in the case of a one-year investment with no compounding.

Using this information, you can apply the formula:

$$ \begin{aligned} &\text{Present Value} = \dfrac{\text{Future Value}}{(1+r)^n}\\ &\textbf{where:}\\ &\text{Future Value} = \text{FV}\\ &r = \text{rate of return}\\ &n = \text{number of periods}\\ \end{aligned} $$

## Present Value Example

Consider a scenario in which you expect to receive a one-time payment of $5,000 in five years. If the discount rate is 8.25%, you want to know what that payment is worth today. The calculation yields: $5,000 / (1 + 0.0825)^5 = $3,363.80.

## Why Does Present Value Matter?

Present value matters because it allows investors and businesses to judge whether a future outcome is worth a present investment. It is especially important when choosing among potential investments that are expected to pay off at different times in the future.

## Conclusion

Present value is a way of expressing the current worth of a future sum of money or stream of cash flows. While useful, it relies on making sound assumptions about future rates of return -- assumptions that become especially complex over longer time horizons.

# What Is Expected Value?
![](https://fastly.jsdelivr.net/gh/bucketio/img9@main/2024/10/20/1729465031968-b3c8959e-1d37-4b8a-91b1-b0b0dfe25143.png)

Expected value (EV) is a term commonly used in the investment industry to denote the anticipated average value of an investment at some point in the future. Investors use expected value to estimate the worth of investments, typically in relative terms against their risk.

By computing expected values, investors can select the scenario most likely to produce the desired outcome. In statistics and probability analysis, expected value is calculated by multiplying each possible outcome by its probability of occurrence, then summing all these values.

### Key Takeaways

- Expected value describes the long-run average level of a random variable based on its probability distribution.
- In investing, the expected value of a stock or other investment is an important consideration and is used in scenario analysis.
- Modern Portfolio Theory uses expected value in conjunction with an investment's risk (standard deviation) to construct optimized portfolios.
- Expected value can help investors assess whether an investment's risk is justified by its potential return.

## Understanding Expected Value

Expected value refers to the anticipated future value of an asset. The expected value of a random variable provides a measure of the center of that variable's distribution. In essence, expected value is the long-run average of the variable.

By the law of large numbers, the average value of a variable converges to its expected value as the number of repetitions approaches infinity. Expected value is also known as the expectation, mean, or first moment.

Expected value can be computed for single discrete variables, single continuous variables, multiple discrete variables, and multiple continuous variables. For continuous variables, integration is required.

Scenario analysis is one technique for computing the expected value of an investment opportunity. It uses estimated probabilities and multivariate models to examine possible outcomes of an investment. Scenario analysis also helps investors assess whether the level of risk they are taking is appropriate given the expected outcome.

**Important:** The difference between expected value and arithmetic mean is that expected value involves a probability distribution, whereas arithmetic mean involves the distribution of occurrences.

## Expected Value Formula

The formula for expected value is:

$$ \begin{aligned} EV=\sum P(X_i)\times X_i\end{aligned} $$

where:

- X is the random variable
- Xi is a specific value of X
- P(Xi) is the probability that Xi occurs

Thus, the expected value of a random variable X is the sum of the product of each value of the random variable and its probability.

## Expected Value in Portfolio Construction

When constructing an investment or financial portfolio, investors need to understand several key factors, including how the assets work and their associated risks. Investors should also be clear about their financial situation, investment objectives, and time horizon.

Once these factors are well understood, investors and their financial advisors can use expected value to construct portfolios that maximize returns while minimizing risk.

You can use expected value to determine the potential return on an investment and thereby choose which assets to include in your portfolio to match your return preferences.

For example, to calculate expected value, first multiply the probability of a positive outcome by the potential return. Suppose an investment has a 60% chance of appreciating by $10,000: 0.6 x $10,000 = $6,000.

Then multiply the probability of a negative outcome by the potential loss. Suppose the investment also has a 40% chance of depreciating by $5,000: 0.4 x $5,000 = $2,000.

Finally, subtract the second result from the first: $6,000 - $2,000 = $4,000. This is the expected value of the investment.

You can then compare two or more investments of interest by following the same steps and comparing expected returns. This can help guide portfolio construction decisions.

Keep in mind that different asset classes have different expected values. That is, the expected value (and risk profile) of a stock differs from that of a bond or an exchange-traded fund (ETF). Calculating and comparing expected values across various assets of interest can be highly informative.

You can also use expected value after a portfolio has been constructed to make adjustments. For example, compare expected values to decide whether to sell a poorly performing asset with no upside potential and replace it with one that has a higher expected value.

## Expected Value Example

To calculate the expected value of a single discrete random variable, multiply each value of the variable by its probability of occurrence.

Consider an ordinary six-sided die. When you roll it, there is a one-in-six chance of landing on 1, 2, 3, 4, 5, or 6. Given this information, the calculation is:

$$ \begin{aligned}\left(\frac{1}{6}\times1\right)&+\left(\frac{1}{6}\times2\right)+\left(\frac{1}{6}\times3\right)\\&+\left(\frac{1}{6}\times4\right)+\left(\frac{1}{6}\times5\right)+\left(\frac{1}{6}\times6\right)=3.5\end{aligned} $$

If you roll a six-sided die an infinite number of times, the average value will converge to 3.5.

## What Is the Expected Value of a Dividend Stock?

A stock's expected value is estimated as the net present value (NPV) of all its future dividends. If you can estimate the dividend growth rate, you can use the dividend discount model (such as the Gordon Growth Model) to predict how much investors would be willing to pay for the stock. However, note that this is a different formula from the statistical expected value discussed in this article.

## How Do You Find the Expected Value of a Non-Dividend Stock?

For non-dividend stocks, analysts often use a multiples approach to derive an expected value. For example, the price-to-earnings (P/E) ratio is frequently used and compared with industry peers. If the technology sector has an average P/E of 25x, then a tech stock's expected value would be 25 times its earnings per share. Again, this differs from the statistical expected value discussed here, but it is another commonly used way to assess a stock's worth.

## How Is Expected Value Used in Portfolio Theory?

Modern Portfolio Theory and related models use mean-variance optimization to derive the optimal risk-adjusted portfolio allocation. Risk is measured as the portfolio's standard deviation, while the mean is the portfolio's expected value (expected return). This indeed leverages the concepts discussed in this article.

## Conclusion

Understanding the concept of expected value is essential for investors. It can help them determine the level of return an investment is likely to generate. Expected value combined with scenario analysis can provide insight into the risk-return tradeoff of an investment, helping investors decide whether to include it in their portfolio.
